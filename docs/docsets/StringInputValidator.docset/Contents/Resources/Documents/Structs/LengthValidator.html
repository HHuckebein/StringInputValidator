LengthValidator

public struct LengthValidator: StringValidator, Equatable

LengthValidator can return several results, e.g. for cases where the characters count is greater than zero * .lengthMismatch - if the count is unequal to lengthBorder * .lengthExceeded - if the count is greater than lengthBorder

The last two ValidatorResults might appear in combination.
